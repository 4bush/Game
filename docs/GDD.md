Game Concept
Star Defender is a fast-paced 2D vertical scrolling space shooter in the endless runner genre. The player controls a spaceship that constantly moves upward through infinite space, dodging waves of enemies, destroying them with different weapons, and collecting power-ups to upgrade the ship. The core mechanic is continuous shooting combined with precise vertical and horizontal maneuvering within the screen boundaries. The win condition is to survive as long as possible and achieve the highest score. The lose condition occurs when the player’s spaceship health reaches zero. The game is developed for Desktop using the libGDX framework.
