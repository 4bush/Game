# Survivor arena
Простая 2D игра в жанре survivor
Используемые дизайн-паттерны:
- Singleton — GameManager, AudioManager, SpawnManager
- Factory — создание разных типов врагов и оружия
- Prototype — клонирование врагов или предметов для спавна
- Decorator — система улучшений/баффов для оружия или игрока (FireDecorator, SpeedDecorator и т.д.)
- Command — система команд для атак, способностей (можно отменять или ставить в очередь)

Цель проекта: демонстрация применения паттернов в реальной игровой механике.
